2020-04-13еҘҪдёңиҘҝе°ұеҫ—еӨ§дјҷдёҖиө·зңӢ21 еҲҶй’ҹ иҜ»е®Ң (еӨ§зәҰ 3146 дёӘеӯ—)

Java 8 ж–°зү№жҖ§

Java 8 (еҸҲз§°дёә jdk 1.8) жҳҜ Java иҜӯиЁҖејҖеҸ‘зҡ„дёҖдёӘдё»иҰҒзүҲжң¬гҖӮ Oracle е…¬еҸёдәҺ 2014 е№ҙ 3 жңҲ 18 ж—ҘеҸ‘еёғ Java 8 пјҢе®ғж”ҜжҢҒеҮҪж•°ејҸзј–зЁӢпјҢж–°зҡ„ JavaScript еј•ж“ҺпјҢж–°зҡ„ж—Ҙжңҹ APIпјҢж–°зҡ„Stream API зӯүгҖӮ

synchronized

synchronized жҳҜ Java дёӯзҡ„е…ій”®еӯ—пјҢжҳҜеҲ©з”Ёй”Ғзҡ„жңәеҲ¶жқҘе®һзҺ°еҗҢжӯҘзҡ„гҖӮ

дј—жүҖе‘ЁзҹҘ synchronized е…ій”®еӯ—жҳҜи§ЈеҶіе№¶еҸ‘й—®йўҳеёёз”Ёи§ЈеҶіж–№жЎҲпјҢжңүд»ҘдёӢдёүз§ҚдҪҝз”Ёж–№ејҸ:

  • еҗҢжӯҘжҷ®йҖҡж–№жі•пјҢй”Ғзҡ„жҳҜеҪ“еүҚеҜ№иұЎгҖӮ
  • еҗҢжӯҘйқҷжҖҒж–№жі•пјҢй”Ғзҡ„жҳҜеҪ“еүҚ Class еҜ№иұЎгҖӮ
  • еҗҢжӯҘеқ—пјҢй”Ғзҡ„жҳҜ () дёӯзҡ„еҜ№иұЎгҖӮ
йҳ…иҜ»жӣҙеӨҡ
2020-04-12еҘҪдёңиҘҝе°ұеҫ—еӨ§дјҷдёҖиө·зңӢВ /В дёҚжҮӮе°ұжҹҘ4 еҲҶй’ҹ иҜ»е®Ң (еӨ§зәҰ 671 дёӘеӯ—)

еҸҢйҮҚжЈҖжҹҘй”Ғе®ҡжЁЎејҸ

еҸҢйҮҚжЈҖжҹҘй”Ғе®ҡжЁЎејҸ(Double checked locking)жҳҜиҪҜд»¶и®ҫи®Ўзҡ„е°ҸжҠҖе·§пјҢз¬¬дёҖйҮҚжЈҖжҹҘи·іиҝҮеӨ§еӨҡж•°дёҚйңҖиҰҒз«һдәүзҡ„жғ…еҶөпјҢд»ҺиҖҢеҮҸе°‘е№¶еҸ‘зі»з»ҹдёӯзҡ„з«һдәүејҖй”ҖгҖӮе®ғз»Ҹеёёиў«з”ЁеңЁеңЁвҖңжғ°жҖ§еҲқе§ӢеҢ–вҖқ (lazy initialization) дёӯпјҢдҫӢеҰӮе®һзҺ°дёҖдёӘзәҝзЁӢе®үе…Ёзҡ„еҚ•дҫӢгҖӮ

йҳ…иҜ»жӣҙеӨҡ

volatile

з”ұдәҺ CPU зј“еӯҳзҡ„еӯҳеңЁеҸҠзј–иҜ‘еҷЁ/CPU зҡ„жҢҮд»ӨйҮҚжҺ’пјҢеӨҡзәҝзЁӢзЁӢеәҸзҡ„иҝҗиЎҢз»“жһңе®һйҷ…дёҠеҫҲйҡҫйў„жңҹпјҢvolatile е…ій”®иҜҚжҳҜ JVM жҸҗдҫӣзҡ„дҝқиҜҒеҸҳйҮҸвҖңеҸҜи§ҒжҖ§вҖқзҡ„жӯҰеҷЁгҖӮJVM дјҡеҒҡи®ёеӨҡеә•еұӮзҡ„е·ҘдҪңжқҘдҝқиҜҒиҝҷз§ҚеҸҜи§ҒжҖ§гҖӮ

йҳ…иҜ»жӣҙеӨҡ
2020-04-12еҘҪдёңиҘҝе°ұеҫ—еӨ§дјҷдёҖиө·зңӢВ /В дёҚжҮӮе°ұжҹҘ5 еҲҶй’ҹ иҜ»е®Ң (еӨ§зәҰ 743 дёӘеӯ—)

еҚ•дҫӢжЁЎејҸ

еҚ•дҫӢжЁЎејҸпјҢд№ҹеҸ«еҚ•еӯҗжЁЎејҸпјҢжҳҜдёҖз§Қеёёз”Ёзҡ„иҪҜд»¶и®ҫи®ЎжЁЎејҸпјҢеұһдәҺеҲӣе»әеһӢжЁЎејҸзҡ„дёҖз§ҚгҖӮеңЁеә”з”ЁиҝҷдёӘжЁЎејҸж—¶пјҢеҚ•дҫӢеҜ№иұЎзҡ„зұ»еҝ…йЎ»дҝқиҜҒеҸӘжңүдёҖдёӘе®һдҫӢеӯҳеңЁгҖӮи®ёеӨҡж—¶еҖҷж•ҙдёӘзі»з»ҹеҸӘйңҖиҰҒжӢҘжңүдёҖдёӘзҡ„е…ЁеұҖеҜ№иұЎпјҢиҝҷж ·жңүеҲ©дәҺжҲ‘д»¬еҚҸи°ғзі»з»ҹж•ҙдҪ“зҡ„иЎҢдёәгҖӮжҜ”еҰӮеңЁжҹҗдёӘжңҚеҠЎеҷЁзЁӢеәҸдёӯпјҢиҜҘжңҚеҠЎеҷЁзҡ„й…ҚзҪ®дҝЎжҒҜеӯҳж”ҫеңЁдёҖдёӘж–Үд»¶дёӯпјҢиҝҷдәӣй…ҚзҪ®ж•°жҚ®з”ұдёҖдёӘеҚ•дҫӢеҜ№иұЎз»ҹдёҖиҜ»еҸ–пјҢз„¶еҗҺжңҚеҠЎиҝӣзЁӢдёӯзҡ„е…¶д»–еҜ№иұЎеҶҚйҖҡиҝҮиҝҷдёӘеҚ•дҫӢеҜ№иұЎиҺ·еҸ–иҝҷдәӣй…ҚзҪ®дҝЎжҒҜгҖӮиҝҷз§Қж–№ејҸз®ҖеҢ–дәҶеңЁеӨҚжқӮзҺҜеўғдёӢзҡ„й…ҚзҪ®з®ЎзҗҶгҖӮ

йҳ…иҜ»жӣҙеӨҡ
2020-04-12еҘҪдёңиҘҝе°ұеҫ—еӨ§дјҷдёҖиө·зңӢВ /В дёҚжҮӮе°ұжҹҘ2 еҲҶй’ҹ иҜ»е®Ң (еӨ§зәҰ 306 дёӘеӯ—)

еҺҹеһӢжЁЎеһӢ

еҺҹеһӢжЁЎејҸжҳҜеҲӣе»әеһӢжЁЎејҸзҡ„дёҖз§ҚпјҢе…¶зү№зӮ№еңЁдәҺйҖҡиҝҮвҖңеӨҚеҲ¶вҖқдёҖдёӘе·Із»ҸеӯҳеңЁзҡ„е®һдҫӢжқҘиҝ”еӣһж–°зҡ„е®һдҫӢ,иҖҢдёҚжҳҜж–°е»әе®һдҫӢгҖӮиў«еӨҚеҲ¶зҡ„е®һдҫӢе°ұжҳҜжҲ‘д»¬жүҖз§°зҡ„вҖңеҺҹеһӢвҖқпјҢиҝҷдёӘеҺҹеһӢжҳҜеҸҜе®ҡеҲ¶зҡ„гҖӮ

еҺҹеһӢжЁЎејҸеӨҡз”ЁдәҺеҲӣе»әеӨҚжқӮзҡ„жҲ–иҖ…иҖ—ж—¶зҡ„е®һдҫӢпјҢеӣ дёәиҝҷз§Қжғ…еҶөдёӢпјҢеӨҚеҲ¶дёҖдёӘе·Із»ҸеӯҳеңЁзҡ„е®һдҫӢдҪҝзЁӢеәҸиҝҗиЎҢжӣҙй«ҳж•ҲпјӣжҲ–иҖ…еҲӣе»әеҖјзӣёзӯүпјҢеҸӘжҳҜе‘ҪеҗҚдёҚдёҖж ·зҡ„еҗҢзұ»ж•°жҚ®гҖӮ

йҳ…иҜ»жӣҙеӨҡ
2020-04-12еҘҪдёңиҘҝе°ұеҫ—еӨ§дјҷдёҖиө·зңӢВ /В дёҚжҮӮе°ұжҹҘ4 еҲҶй’ҹ иҜ»е®Ң (еӨ§зәҰ 566 дёӘеӯ—)

еҜ№иұЎжұ жЁЎејҸ

еҜ№иұЎжұ пјҲиӢұиҜӯпјҡobject pool patternпјүжҳҜдёҖз§Қи®ҫи®ЎжЁЎејҸгҖӮдёҖдёӘеҜ№иұЎжұ еҢ…еҗ«дёҖз»„е·Із»ҸеҲқе§ӢеҢ–иҝҮдё”еҸҜд»ҘдҪҝз”Ёзҡ„еҜ№иұЎпјҢиҖҢеҸҜд»ҘеңЁжңүйңҖжұӮж—¶еҲӣе»әе’Ңй”ҖжҜҒеҜ№иұЎгҖӮжұ зҡ„з”ЁжҲ·еҸҜд»Ҙд»Һжұ еӯҗдёӯеҸ–еҫ—еҜ№иұЎпјҢеҜ№е…¶иҝӣиЎҢж“ҚдҪңеӨ„зҗҶпјҢе№¶еңЁдёҚйңҖиҰҒж—¶еҪ’иҝҳз»ҷжұ еӯҗиҖҢйқһзӣҙжҺҘй”ҖжҜҒе®ғгҖӮиҝҷжҳҜдёҖз§Қзү№ж®Ҡзҡ„е·ҘеҺӮеҜ№иұЎгҖӮ

иӢҘеҲқе§ӢеҢ–гҖҒе®һдҫӢеҢ–зҡ„д»Јд»·й«ҳпјҢдё”жңүйңҖжұӮйңҖиҰҒз»Ҹеёёе®һдҫӢеҢ–пјҢдҪҶжҜҸж¬Ўе®һдҫӢеҢ–зҡ„ж•°йҮҸиҫғе°‘зҡ„жғ…еҶөдёӢпјҢдҪҝз”ЁеҜ№иұЎжұ еҸҜд»ҘиҺ·еҫ—жҳҫи‘—зҡ„ж•ҲиғҪжҸҗеҚҮгҖӮд»Һжұ еӯҗдёӯеҸ–еҫ—еҜ№иұЎзҡ„ж—¶й—ҙжҳҜеҸҜйў„жөӢзҡ„пјҢдҪҶж–°е»әдёҖдёӘе®һдҫӢжүҖйңҖзҡ„ж—¶й—ҙжҳҜдёҚзЎ®е®ҡгҖӮ

йҳ…иҜ»жӣҙеӨҡ
2020-04-11еҘҪдёңиҘҝе°ұеҫ—еӨ§дјҷдёҖиө·зңӢВ /В дёҚжҮӮе°ұжҹҘ3 еҲҶй’ҹ иҜ»е®Ң (еӨ§зәҰ 428 дёӘеӯ—)

иҝҷйҖҡеёёжҳҜд»ҘдёҖдёӘж——еҸ·жқҘе®һзҺ°пјҢз”Ёж——еҸ·жқҘж ҮзӨәжҳҜеҗҰе®ҢжҲҗе…¶зЁӢејҸгҖӮжҜҸж¬ЎиҜ·жұӮеҜ№иұЎж—¶пјҢдјҡе…ҲжөӢиҜ•жӯӨж——еҸ·гҖӮеҰӮжһңе·Іе®ҢжҲҗпјҢзӣҙжҺҘдј еӣһпјҢеҗҰеҲҷеҪ“еңәжү§иЎҢгҖӮ
