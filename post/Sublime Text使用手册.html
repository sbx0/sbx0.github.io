Sublime Text

Sublime Text 是一款流行的代码编辑器软件，也是HTML和散文先进的文本编辑器，可运行在Linux，Windows和Mac OS X。也是许多程序员喜欢使用的一款文本编辑器软件。

经常误入Command Mode

这是因为在Sublime Text中进行选中操作中，如果使用了Esc退出选中状态，会进入Command Mode。所谓的Command Mode是因为开启了Vintage，也就是Vim模式。可以按a、i或者o回到Insert Mode。但是由于经常误入Command Mode，很难受，所以在设置中直接关闭Vim模式。

"ignored_packages":
[
	"Vintage"
]

快捷键修改

IntelliJ IDEA的ctrl+d（复制当前行到下一行）非常好用，但是在Sublime Text中对应的快捷键为ctrl+shift+d，所以要修改一下快捷键配置。

[
	{ "keys": ["ctrl+d"], "command": "duplicate_line" },
	{ "keys": ["ctrl+shift+d"], "command": "find_under_expand" },
]
